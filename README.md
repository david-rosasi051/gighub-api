# GigHub Nairobi Freelance Gigs API

A FastAPI-based REST API for managing freelance gigs in Nairobi.  
Built as part of coursework project **C027-01-0922/2024**, this API provides endpoints to create, read, update, delete, and search freelance gigs across categories like Development, Design, and Writing.


## 🚀 Features
- **List gigs** with optional filters (category, min/max budget).
- **Search gigs** by title.
- **Retrieve gig by ID** with error handling.
- **Create gigs** with validation (title, description, category, budget, client name).
- **Update gigs** (budget and status).
- **Delete gigs** by ID.
- **Preloaded dataset** of 7 sample gigs in KES currency.
